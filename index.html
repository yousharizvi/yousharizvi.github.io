Prerequisite

HTML

Although ionic's HTML is said to be "INVALID HTML" as it does not conforms with the the tags given in HTML5, but it's structure and syntax is just same as that of valid HTML5.

JavaScript (Must know angular)

If you've worked on JavaScript, specially AngularJS and you know about OOP which is must for a dev, then it's going to be pretty easier to be an ionic developer or simply write ionic clients and build for iOS, Android or WP platforms.

CSS (SASS/SCSS Knowledge is a plus)

CSS is a prerequisite but no worries if you have no experience with SASS or SCSS because you can also go with simple CSS3. But lets know a bit about SCSS here.

Whats new?

Typescript

For me, typescript is just typed JavaScript but with all the latest elements of JavaScript(ES6) involved. Learn more about typescript here. Important is to read about types and generics, but it really isn't a hurdle in the way to write an app.

Install
$ npm install -g typescript
Compile
$ tsc <file_name>.ts
Useful Examples
// Setting types of a variable
let a: string; 

// Arrays, Objects and the 'any'
let arr: Array<string>; // Declares an array of strings 
let obj: object; // Declares an object

Getting Started

Getting friendly with the ionic cli

Install ionic cli
$ npm install -g ionic
On Mac and Linux, youâ€™ll need to add sudo to install Ionic globally
$ sudo npm install -g ionic
Initialize a project
$ ionic start <project_name> <template (otional)> tutorial --v2
Generate a page
$ ionic generate <page_name>
Generate a service
$ ionic generate <service_name>
$ ionic g <service_name> // or simply 'g'

Ionic generator

Ionic cli provides a generator function to create pages and services for your app
  • Page (Component)
  • Provider (Service)
Examples
$ ionic start myApp --v2 
$ ionic generate myPage 
$ ionic generate authService 
$ ionic serve
Ionic serve is intended to run the app in the browser, it has the livereload. Check serve documentation
Some useful tips: 
$ ionic serve -l // serves in ionic lab

Pages (Components)

Components can also be called views, which are easy to create and use or reuse. A sample component has following structure. 

myComponent/
  |__ myComponent.component.ts 
  |__ myComponent.component.scss 
  |__ myComponent.component.html 
  |__ myComponent.component.spec.ts

Providers (Services)

Provider or a service is just a typescript class exported as service. Methods are useful to work as central control over the application.
